#### Запуск

- Запуск проекта:

    `yarn start`

- Запуск проекта в режиме разработки

    `yarn dev`

#### Сборка

- Сборка проекта:

    `yarn build`

#### Тесты

- Запуск теста в командной строке

    `yarn build && yarn cypress:test`

- Запуск тестов в графическом интерфейсе

    `yarn cypress`
